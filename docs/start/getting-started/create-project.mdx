# 在 Arduino IDE 中创建 ESP32 范例程序

现在，你已经安装了 ESP32 的开发环境，接下来我们将介绍如何在 **Arduino IDE** 中创建一个新的 ESP32 项目，并编写一个 **LED 闪烁程序**。

---

## 1. 打开 Arduino IDE 并创建新项目

1. **打开 Arduino IDE**。
2. 点击 **文件 > 新建**，创建一个新的 Arduino 项目。
3. 在 **工具 > 开发板** 中，选择 **ESP32 设备**（例如 `DOIT ESP32 DEVKIT V1`）。
4. 选择 **工具 > 端口**，确保端口名称类似于 `COMX`（Windows）或 `/dev/cu.SLAB_USBtoUART`（Mac）。


---

## 2. 编写 ESP32 示例代码

在代码编辑窗口中，输入以下 **LED 闪烁（Blink）示例代码**，用于控制 ESP32 开发板上的 LED 进行 **1 秒亮、1 秒灭** 的循环闪烁：

```cpp title="ESP32_Blink.ino" showLineNumbers
// 定义 LED 引脚（ESP32 DevKit V1 通常使用 GPIO2）
const int LED_PIN = 2;

void setup() {
    pinMode(LED_PIN, OUTPUT); // 设置 LED 引脚为输出模式
}

void loop() {
    digitalWrite(LED_PIN, HIGH); // 亮灯
    delay(1000); // 等待 1 秒
    digitalWrite(LED_PIN, LOW); // 关灯
    delay(1000); // 等待 1 秒
}
```
